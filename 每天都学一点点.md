typeof: 检测基本数据类型的值，所以出现了instanceof来检测引用类型

```
function foo(){
	console.log(2);
}
typeof foo;   //"function"
```



标准Http协议支持六种请求方法，即：

1、GET

2、POST

3、PUT 从客户端向服务器传送的数据取代指定的文档的内容。

4、Delete  请求server删除页面

5、HEAD 跟get类似，只请求头部信息

6、Options 获取服务器支持的HTTP请求方法；用来检查服务器的性能。
